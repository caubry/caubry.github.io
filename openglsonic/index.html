Project description

This video shows a - 3d Computer Graphics project for my Masters degree in Computer Games Technology at City University London - project created using C++, OpenGL 4.0 and GLSL.

Demonstrating the uses of fragment and vertex shaders, as well as some advanced rendering techniques such as fog, toon shaders and camera shake.

Finally, the track has been generated programmatically, using OpenGL primitives and Catmull-Rom splines.
