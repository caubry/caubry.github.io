Project description

This project uses the Bullet Physics engine and implement an AI algorithm illustrating flocking. Three simple steering behaviours have to be applied to describe how an individual boid maneuvers: separation, alignment and cohesion. Regarding the bird behaviours, three physical principles have been followed: thurst, lift and drag.

Furthermore, the flock is always trying to avoid static obstacles placed across the world, while the camera is finding the leader of the flock in order to follow it.
